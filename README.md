# 🎬 MovieMate

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-Frontend-646CFF?logo=vite)

![Status](https://img.shields.io/badge/Status-Active-success)

MovieMate is a modern movie discovery web application built with **React**, **Vite**, **React Router**, and **TailwindCSS**.  
It is designed for fast performance, clean UI, and smooth navigation while browsing movies.

---

## 🚀 Features

- 🎥 Browse movies
- 🔍 Search functionality
- 🔍 Watch list functionality
- 📄 Detailed movie pages
- ⚡ Fast development with Vite
- 🎨 Responsive UI with TailwindCSS
- 🔄 Client-side routing with React Router

---

## 🧰 Tech Stack

- React 19
- Vite
- React Router DOM
- TailwindCSS v4
- clsx

---

## 🛠️ Getting Started

Follow these steps carefully to set up the project from scratch.

---

## 1️⃣ Create a new React project with Vite

```bash
npm create vite@latest my-app
```

## 2️⃣ Navigate into the project directory

```bash
cd my-app
```

## 3️⃣ Install base dependencies

```bash
npm install
```

4️⃣ Install project dependencies

```bash
npm install react react-dom react-router-dom clsx tailwindcss @tailwindcss/vite
```

5️⃣ Start the development server

```bash
npm run dev
```
